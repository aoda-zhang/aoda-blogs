## Migrate to pnpm ( optional,recommended )

If you are a JavaScript developer, you will know we have to choose a package manager to install our project
dependencies. I bet that most people used npm to be your first package manager, yes i did that too, but as time goes
by,i found it tooks more time to install with the project grow. i tried to find out a better one, yes,that is <b>pnpm</b>,it is really really 
faster than npm, mean while , the node_modules size is become less,plus, you can still use npm command to run pnpm. I will highly recommend you give a try for pnpm.
it is time to consider to replace your package manager.I already have used pnpm 1 year, so far so good, i feeled luckly to choose it. <br/>

<b>Just run this command to install pnpm globally if you want to migrate to pnpm from npm</b>

```sh
npm install -g pnpm
```

More about pnpm ,please refer to [here](https://pnpm.io).

## Create a new React project by Vite.

### What is Vite ?

This is Vite offical desLVite is a build tool that aims to provide a faster and leaner development experience for modern web projects. ()

### Why creat React project by Vite ?

If your app has unusual constraints not served well by these frameworks, or you prefer to solve these problems yourself, you can roll your own custom setup with React. Grab react and react-dom from npm, 
set up your custom build process with a bundler like Vite or Parcel, 
and add other tools as you need them for routing, static generation or server-side rendering, and more.

More about Vite, please refer to [here](https://vitejs.dev).

<b>Just run this command to create your first React + Vite project</b>

```sh
pnpm create vite fullStack-React --template react-swc-ts
```
